NBA Game Performance Analytics

This project ingests NBA game logs, cleans the data, and generates advanced metrics like TS%, eFG%, Game Score, and rolling averages to highlight top performers and trends.

  • Built with Python (pandas, matplotlib).
  • Produced player trend charts and top-performance tables.
  • Automated workflow using GitHub Actions.
