# Frontend Designer Portfolio

This repository contains my professional **Frontend Designer Portfolio**, a live website showcasing only my **frontend-related projects**.  
It demonstrates my skills in building modern, responsive, and accessible web experiences with a focus on **UI/UX design systems**.

---

## ✨ Key Features
- 🎨 **Modern UI/UX** — Clean interface with a consistent design system.  
- 📱 **Responsive Design** — Mobile-first layouts optimized for all devices.  
- ♿ **Accessibility** — Semantic HTML, ARIA roles, and WCAG-conscious contrast.  
- 🔎 **Frontend Projects Only** — Automatically filters and displays my GitHub repositories that focus on frontend work.  
- ⚡ **Lightweight & Fast** — Built with HTML, CSS, and vanilla JavaScript.  

---

## 🛠️ Tech Stack
- **Languages:** HTML5, CSS3, JavaScript (ES6+)  
- **Styling & Layout:** Custom CSS, responsive grid system  
- **Design Tools:** Figma, UI prototyping, design systems  
- **Version Control:** Git & GitHub  
- **API:** GitHub REST API (frontend repos filter)  
- **Deployment:** GitHub Pages  

---

## 📂 Project Structure
```
├── index.html          # Portfolio homepage
├── style.css           # Styling and theme system
├── script.js           # Fetches and renders frontend-only repos
└── README.md           # Documentation
```

---

## 🚀 Deployment
1. Clone this repository:
   ```bash
   git clone https://github.com/YSayaovong/portfolio.git
   ```
2. Push to your own GitHub account.  
3. Enable **GitHub Pages** in your repository settings.  
4. Your live portfolio will be available at:  
   ```
   https://ysayaovong.github.io/portfolio
   ```
